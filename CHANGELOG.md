# Changelog

## v0.3.1 - 2026-07-18

- Replaced the app, tray, shortcut, and installer icon with the simplified focus/recovery clock mark.
- Fixed startup recovery so exiting the app ends the current focus task instead of resuming it later or triggering a forced break.

## v0.3.0 - 2026-07-13

- Rebuilt the full interface as a bilingual cyber command center with consistent Focus, Insights, Squad, and Armory views.
- Added task intentions, focus/recovery analytics, weekly share cards, onboarding, and optional privacy-first squads.
- Added restart-safe sessions, full-body persona framing, and locked preset selection during active training.
- Simplified the Windows tray: left-click opens Focus, while right-click shows only valid actions and accurate page links.

## v0.2.0

- Rebuilt the app around Focus, Insights, Squad, and Armory views.
- Added first-run setup, task intentions, reduced-motion support, and weekly share cards.
- Added versioned session history, balanced recovery scoring, and restart-safe timer state.
- Added an optional Supabase squad backend with encrypted local auth storage and row-level security.

## v0.1.0

Initial public release.

- Windows tray-first Pomodoro timer.
- Bilingual Chinese / English UI.
- Three focus presets: 20 / 40 / 60 minutes.
- One-minute pre-break warning window.
- Full-screen always-on-top break overlays.
- Post-break decision gate.
- Emergency bypass with password and required reason.
- Local settings and history.
- Persona rotation and discipline score.
- Extensible Overlay Pack architecture.
- Windows installer and portable zip packaging.
